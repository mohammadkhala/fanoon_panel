## محتوى مقترح لكل صفحة (ملخص)

| الصفحة | الخطوات المقترحة |
|--------|-------------------|
| **تعديل منتج** | 1) عدّل الحقول المطلوبة 2) الخصائص والمتغيرات 3) الصور 4) الترجمة 5) أسعار أنواع العملاء |
| **استيراد جماعي** | 1) حمّل الملف النموذجي 2) املأ البيانات 3) ارفع الملف 4) راجع المنتجات وأضف الصور |
| **إضافة تصنيف** | 1) الاسم والوصف 2) الصورة 3) التصنيف الأب (للفرعي) |
| **قائمة الطلبات** | 1) تصفية بالتاريخ 2) تصفية بالحالة 3) التصدير 4) البحث |
| **تفاصيل الطلب** | 1) تغيير الحالة 2) طباعة الفاتورة 3) إضافة مندوب |
| **أنواع العملاء** | 1) أضف نوعاً 2) حدد الاسم 3) استخدمه في الأسعار |
| **المدن** | 1) أضف مدينة 2) أضف المناطق 3) رسوم التوصيل |
| **إعدادات الأعمال** | 1) وضع الصيانة 2) الشعارات 3) اللغات 4) برنامج الولاء |
