## 2. ما تم إنجازه جزئياً أو بفجوات

### المرحلة 2 — صفحة مركز المساعدة
| المهمة | الحالة | التفاصيل |
|--------|--------|----------|
| تصميم الصفحة: عنوان، وصف، أقسام | ⚠️ | الصفحة الحالية تعرض بطاقة بسيطة فقط |
| روابط سريعة: الطلبات، المنتجات، التصنيفات، الإعدادات | ❌ | غير موجودة في الـ view |
| نص ترحيبي، نظرة عامة، خطوات البدء | ❌ | الترجمات موجودة (help_step_1..6، help_getting_started_text) لكنها **لا تُعرض** في الصفحة |

**الفجوة:** ملف `help/index.blade.php` لا يستخدم المحتوى المترجم. الصفحة تعرض فقط:
```blade
<p class="mb-0">{{ translate('user_guide') }}</p>
```
