## 4. تفاصيل تنفيذ كل مهمة

### 4.1 تصدير قائمة العملاء إلى Excel

| الخطوة | الملف | الإجراء |
|--------|-------|---------|
| 1 | `routes/admin.php` | إضافة `Route::get('export', ...)` |
| 2 | `CustomerController.php` | إضافة دالة `exportCustomers()` |
| 3 | `customer/list.blade.php` | إضافة زر "تصدير" |
| 4 | استخدام `FastExcel` | مثل `OrderController::exportOrders` |

### 4.2 فلتر التقييم للمنتجات

| الخطوة | الملف | الإجراء |
|--------|-------|---------|
| 1 | `ProductController::list()` | إضافة معامل `rating_min` |
| 2 | Query | `whereHas('reviews')` + `havingRaw('avg(rating) >= ?')` أو جدول فرعي |
| 3 | `product/list.blade.php` | إضافة select للتصفية حسب التقييم (مثلاً: 4+، 3+، الكل) |

### 4.3 شارات مراجعة للعملاء

| الخطوة | الملف | الإجراء |
|--------|-------|---------|
| 1 | تحديد المعيار | مثلاً: عميل لديه 5+ طلبات = "مشتري مؤكد" |
| 2 | `Customer` model أو View | إضافة badge/شارة حسب المعيار |
| 3 | `customer/view.blade.php` أو القائمة | عرض الشارة |

### 4.4 تقسيم العملاء

| الخطوة | الإجراء |
|--------|---------|
| 1 | إضافة فلتر "نوع العميل" أو "شريحة": VIP، متكرر، جديد، غير نشط |
| 2 | حساب: إجمالي المشتريات، عدد الطلبات، تاريخ آخر طلب |
| 3 | عرض الفلاتر في قائمة العملاء |

### 4.5 طباعة جماعية للفواتير

| الخطوة | الإجراء |
|--------|---------|
| 1 | في قائمة الطلبات: checkbox لاختيار طلبات متعددة |
| 2 | زر "طباعة المحدد" |
| 3 | فتح نوافذ متعددة أو إنشاء PDF واحد يجمع الفواتير |

### 4.6 بحث موحّد

| الخطوة | الإجراء |
|--------|---------|
| 1 | حقل بحث في الهيدر أو الشريط الجانبي |
| 2 | Route + Controller للبحث في: الطلبات، المنتجات، العملاء |
| 3 | صفحة نتائج أو dropdown يعرض النتائج مصنّفة |

## 5. ملخص تنفيذي

| الفئة | تم | لم يتم |
|-------|-----|--------|
| **المنتجات** | وسوم، منتجات ذات صلة، فلاتر أساسية (سعر، مخزون، وسم...) | فلتر التقييم، بحث بالخصائص |
| **الطلبات** | سجل التغييرات | طباعة جماعية |
| **العملاء** | قائمة، فلاتر أساسية | تصدير Excel، شارات، تقسيم |
| **لوحة التحكم** | — | بحث موحّد |

**التوصية للنسخة الأولى:** تنفيذ المرحلة أ (تصدير العملاء، فلتر التقييم، شارات) — جهد ~6–9 ساعات.
