# معيار موحد للجداول - هوية بصرية

## المرجع
جدول **قائمة الزبائن** (`admin-views/customer/list.blade.php`)

## البنية القياسية

```html
<div class="table-responsive datatable-custom">
    <table class="table table-hover table-border table-thead-bordered table-nowrap table-align-middle card-table">
        <thead class="thead-light">
            <tr>
                <th>#</th>
                <th>...</th>
            </tr>
        </thead>
        <tbody>
            ...
        </tbody>
    </table>
</div>
```

## Classes المستخدمة

| Class | الغرض |
|-------|-------|
| `table-responsive datatable-custom` | غلاف التمرير والتنسيق |
| `table` | أساس Bootstrap |
| `table-hover` | تأثير عند المرور على الصف |
| `table-border` | حدود بين الخلايا |
| `table-thead-bordered` | حدود رأس الجدول |
| `table-nowrap` | منع التفاف النص |
| `table-align-middle` | محاذاة عمودية للوسط |
| `card-table` | تنسيق البطاقة |
| `thead-light` | خلفية فاتحة للرأس |
